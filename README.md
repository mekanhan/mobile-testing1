## Installation
```
npm i
```

## Execution
- To run a script:
```
npm run android
npm run ios
```
## Reinstall
- To reinstall dependencies run a script:
```
npm run cleanMac
npm run cleanWindows
```
## Report
- To open HTML report run a script:
```
npm run report
```
